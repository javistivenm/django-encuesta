## Django Template

Simple Django starter configured with SQLite.

### Quick start

```bash
python3 -m venv .venv
source .venv/bin/activate
python3 -m pip install --upgrade pip
python3 -m pip install -r requirements.txt
python3 manage.py migrate
python3 manage.py runserver
```
